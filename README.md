## 🎨 Featured Projects

### [bing-glava-suite (bgs)](https://github.com/Krzysztofci/bing-glava-suite)
**Multi-instance GLava visualization studio for the Linux desktop.**

![Multi-instance GUI demo](https://raw.githubusercontent.com/Krzysztofci/bing-glava-suite/main/screenshots/Demo-1.0.0-MI.gif)

#### 📖 The Story Behind It
This project started as a simple script to download daily Bing wallpapers because existing tools like Variety had annoying bugs for me, and I wanted a clean way to sync my Linux background with what the Starth Bing Wallpaper app did on my Android phone. 

Over time, I started exploring audio visualizers and settled on GLava. I quickly felt the urge to synchronize its colors with the daily wallpaper, which laid the foundation for **bgs**. It just kept growing from there — and I still have plenty of ideas for it.

### [Universal i18n Commander (UniC)](https://github.com/Krzysztofci/Universal-i18n-Commander)
**Lightweight localization manager for JSON-based projects.**

<img src="https://raw.githubusercontent.com/Krzysztofci/Universal-i18n-Commander/main/screenshots/UniC.png" width="700" alt="Universal i18n Commander screenshot">

#### 📖 Why I Built It
As **bgs** grew, managing its localization and translations became complex. I built **UniC** specifically to streamline the development of my own projects, making it easy to scan source code in real time for duplicate or unused "ghost keys". It belongs to a collection of various smaller utilities I've built to optimize my workflow.

*🔧 Built with: Python 3.12, GTK4 (via PyGObject), and JSON-native UI structures.*
